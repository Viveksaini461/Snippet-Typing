How to Use

  • Select a language (HTML, CSS, or JavaScript) and a difficulty level (Beginner, Intermediate, Advanced).
  • Click the "New Snippet" button to load a random code snippet.
  • Start typing the code shown in the box above the text area.
  • The timer starts automatically when you begin typing.
  • Each difficulty level has a different time limit:
    • Beginner: 15 seconds
    • Intermediate: 30 seconds
    • Advanced: 50 seconds
  • Once time is up, typing is locked and your WPM (Words Per Minute) and Accuracy are shown.
  • You can copy the snippet using the 📋 Copy button.
  • Toggle sound effects using the 🔊 / 🔇 button.
  • Switch between Light and Dark mode using the theme toggle in the top bar.
